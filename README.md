## 📝 Project Overview
This project implements and optimizes a **Monocular Depth Estimation (MDE)** system specifically for the **NVIDIA Jetson Orin Nano**. By leveraging a self-supervised deep learning model (Monodepth2) and custom **C++/CUDA** kernels for metric scaling, the system achieves real-time performance suitable for autonomous robotics and ADAS (Advanced Driver Assistance Systems).

<img width="1203" height="677" alt="Picture1" src="https://github.com/user-attachments/assets/e1a3460e-00a0-4fd9-b775-3badb3186e5c" />


### Why Monocular Depth?
Traditional depth sensing relies on expensive LiDAR or Stereo cameras. This project proves that high-accuracy 3D geometry can be extracted from a **single 2D camera**, significantly reducing hardware costs and complexity.

## 🚀 Key Features
* **Deep Learning Model:** Based on Monodepth2 with a ResNet18 encoder, trained on the KITTI dataset.
* **CUDA Optimization:** Custom `scale_app` module written in C++/CUDA to parallelize the metric recovery process.
* **Performance:** Optimized for the Ampere architecture (Orin Nano) with a ~5x speedup over standard PC execution for specific modules.
* **Analytical Dashboard:** Automated generation of depth heatmaps and distance distribution graphs.

## 📂 System Architecture
The system follows a modular pipeline to ensure maximum efficiency on embedded hardware:

1.  **Inference:** `test_simple.py` processes the image through the neural network.
2.  **Metric Scaling:** `scale_app` (CUDA) recovers real-world distances in meters.
3.  **Visualization:** `demonstratie_calitate.py` generates the final output.
